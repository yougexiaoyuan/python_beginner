# **适合 Python 入门开发者**

Python常用于人工智能，数据分析，爬虫，Web开发，GUI客户端开发等。

Python 是一种解释型语言： 开发过程中没有了编译这个环节。类似于PHP。

Python 是交互式语言： 可以在一个 Python 提示符 >>> 后直接执行代码。

Python 是面向对象语言: Python支持面向对象的风格的编程技术。
